How to use:
This can only be played if you have installed the main Pokemon3D game.
After downloading the Pokemon Brown folder, move it to your GameModes folder in the main Pokemon folder.
Once the Pokemon Brown folder is in the GameModes folder, you can launch the game and start a new game.

Reporting bugs/crashes:
If you get a crash or there is a bug that prevents your from doing something, please let us know.
You can let us know via the Pokemon3D forums. Either post the issue on the GameMode thread or send a PM to Leahkim or Miragics.
